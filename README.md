# Hit and Blow Game

This is a classic "Hit and Blow" game implemented as a static web application.

## How to Play

1.  **Start a new game:** A secret 4-digit number is automatically generated when you open the game.
2.  **Guess the number:** Enter your 4-digit guess in the input field and click the "Guess" button.
3.  **Check the result:**
    *   **Hit:** A digit is correct and in the right position.
    *   **Blow:** A digit is correct but in the wrong position.
4.  **Keep guessing:** Continue guessing until you have 4 hits.

## Game Options

This version of the game includes the following options:

*   **Allow Duplicate Numbers:** Check this box to allow the secret number to contain duplicate digits.
*   **Number of Digits to Use:** Choose the number of unique characters to be used in the secret number, from 6 (0-5) to 16 (0-F).

The default settings are a 4-digit number with no duplicate numbers, using the digits 0-9. Changing any of these options will start a new game.
